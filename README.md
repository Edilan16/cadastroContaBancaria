# CadastroContaBancaria

Projeto em Java para simular operações de uma conta bancária via console.

## Funcionalidades
- Criar conta (número e titular)
- Depósito (inclui depósito inicial opcional)
- Saque (com taxa fixa de 5.00)
- Exibir dados/saldo da conta

## Conceitos utilizados
- Programação Orientada a Objetos (POO)
- Construtores
- Encapsulamento (getters/setters)

## Como executar
1. Abra o projeto em uma IDE Java (ex.: Eclipse/IntelliJ).
2. Execute a classe `application.program`.
3. Siga as instruções exibidas no console para criar a conta, depositar e sacar.

## Estrutura
- `src/application/program.java`: classe principal (entrada do programa)
- `src/entities/Account.java`: entidade da conta e regras de depósito/saque
